# TipSplit — Tip Calculator & Bill Splitter

A single-screen, live-updating tip calculator and bill splitter.

## How to Run

No build step needed — it's vanilla HTML/CSS/JS.

```bash
# Option 1: open directly
open index.html        # macOS
start index.html       # Windows

# Option 2: use a local server (avoids any browser quirks)
npx serve .
# then visit http://localhost:3000

# Option 3: VS Code Live Server extension → right-click index.html → Open with Live Server
```
